# Swiss Ephemeris Library

Place the Swiss Ephemeris JAR file here:
- `swisseph-2.10.03.jar`

## Download
You can download the Swiss Ephemeris Java library from:
- Official: https://www.astro.com/swisseph/
- Maven: https://repo1.maven.org/maven2/com/github/virtualwiz/swisseph/

## Alternative
The GitHub Actions workflow will attempt to download this automatically during the build process.
